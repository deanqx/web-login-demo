A demo project about Zero Downtime Deployment.

# Stack
## Frontend
- Sveltekit
- Scss
- Bootstrap

The website contains
- a login
- an admin interface to create new users and edit old ones
- an editable user profile with profile picture upload

## Backend
- Pheonix (Elixir)
- Postgres
- NixOS
- Docker
- Kubernetes

# Deploying

```bash
docker compose build
docker compose up -d
```
